# How to get a Groq API Key

This application uses Groq's super-fast AI infrastructure to power the Study Explainer. To run the app on your computer, you need to generate a free API key.

## Steps to Get Your Free API Key

1. **Visit GroqCloud:** Navigate to [console.groq.com](https://console.groq.com/) in your browser.
2. **Create an Account or Log In:** You can easily sign in using your existing Google account or sign up with an email address.
3. **Go to API Keys:** On the left-hand navigation menu of the Groq console, locate and click on **API Keys**.
4. **Create New Key:** Click on the button labeled **"Create API Key"**.
5. **Name Your Key (Optional):** Give your key a descriptive name, such as `AI-Study-Explainer`.
6. **Copy the Key:** Your new API key will be displayed immediately. It usually starts with `gsk_`. **Copy this key to your clipboard.** 
*(Note: You won't be able to see the full key again once you close this window, so make sure to copy it!)*

## Using the Key in the App

1. Open `index.html` in your web browser.
2. Paste some study text into the main box.
3. Click **"Explain Now"**.
4. The application will trigger a prompt asking for your API Key.
5. Paste the key you copied from Groq console and hit Enter.

The key will be saved safely into your browser's local storage so you don't have to enter it again on that browser!
